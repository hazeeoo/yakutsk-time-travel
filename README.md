# Якутск: Сквозь века

Интерактивный лендинг про историю Якутска от острога 1632 года до современной IT-столицы.

## Что это

Одностраничник с таймлайном, который рассказывает про 7 ключевых эпох города. Сделан на Next.js с кучей анимаций и параллакс-эффектов.

## Запуск

```bash
pnpm install
pnpm dev
```

Откроется на `http://localhost:3000`

## Стек

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS
- Framer Motion (анимации)
- Radix UI (компоненты)
- Lenis (плавный скролл)

## Структура

```
app/              # страницы и layout
components/       # все компоненты
  ui/            # UI-компоненты от shadcn
lib/             # данные и утилиты
public/images/   # картинки и видео
```

## Эпохи

1. **1632** — Основание острога
2. **XIX век** — Купеческая торговля
3. **Начало XX** — Наука и образование
4. **1940-е** — Трасса АлСиб
5. **1970-е** — Строительство на мерзлоте
6. **1990-е** — Возрождение культуры
7. **Сегодня** — IT-хаб

## Фичи

- Видео-фон на главной
- Боковая навигация по таймлайну
- 3D-наклон картинок при наведении
- Статистика и факты для каждой эпохи
- Частицы и ambient-эффекты
- Адаптив под мобилки

## Сборка

```bash
pnpm build
```
